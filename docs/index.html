Introduction

The Persian Validator package is designed to provide comprehensive Persian (Farsi) validation for Laravel applications...

Requirements

  • Laravel 8, 9, 10
  • PHP 7.4, 8

Install

Via composer:

composer require mrtolouei/persia-validator

Or use git clone:

git clone https://github.com/mrtolouei/persia-validator.git

Usage

Example of using the fa_string rule in Laravel validation:

$request->validate([
    'name' => 'fa_string',
]);
